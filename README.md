# SPMB SMK Budi Bakti Ciwidey

Sebuah landing page statis untuk Sistem Penerimaan Murid Baru (SPMB) SMK Budi Bakti Ciwidey tahun ajaran 2026/2027.

## Deskripsi

Halaman ini dibuat untuk mempromosikan program keahlian, fasilitas, dan informasi pendaftaran siswa baru. Menggunakan HTML, CSS kustom, dan Tailwind CSS melalui CDN.

## Struktur Proyek

- `index.html` — halaman utama lengkap dengan hero, program keahlian, fasilitas, pendaftaran, dan modal jurusan.
- `assets/` — menyimpan gambar dan aset pendukung.

## Fitur Utama

- Navigasi responsif dengan menu mobile
- Hero section dengan CTA pendaftaran
- Grid program keahlian dengan overlay interaktif
- Modal detail jurusan untuk PPLG, DKV, dan BRP
- Carousel fasilitas dengan kontrol slide
- Bagian pendaftaran dengan tombol aksi dan tautan sosial

## Teknologi

- HTML5
- Tailwind CSS (via CDN)
- JavaScript vanilla untuk:
  - modal jurusan
  - menu mobile
  - carousel fasilitas
  - efek scroll pada navbar

## Menjalankan Proyek

1. Buka folder proyek di file explorer.
2. Klik dua kali `index.html` untuk membukanya di browser.

## Kustomisasi

- Ubah teks dan konten di `index.html` sesuai kebutuhan.
- Ganti gambar di `assets/img/` jika ingin menggunakan foto atau logo baru.
- Sesuaikan warna dan gaya di blok `<style>` dalam `index.html`.

## Catatan

- Karena menggunakan Tailwind lewat CDN, koneksi internet diperlukan untuk memuat gaya.
- Untuk penggunaan offline penuh, unduh dan simpan file Tailwind secara lokal atau konversi ke CSS statis.

## Lisensi

Proyek ini bisa digunakan dan disesuaikan sesuai kebutuhan sekolah atau promosi pendaftaran.
